# 二、 运算法则

模（Mod）运算的规则、公式与基本四则运算有些类似，但是除法例外。其规则如下：  
![在这里插入图片描述](https://img-blog.csdnimg.cn/20200420161228337.png)  
模运算满足结合律、交换律、分配率，具体如下：

A. 结合律

((a + b) % p + c)% p = (a+(b+c)%p)%p

((a - b) % p * c)% p = (a * (b - c)%p)%p

B. 交换律

(a + b) % p = (b + a) % p

(a - b) % p = (b - a) % p

C. 分配率

(a + b) % p = (a % p + b % p) % p

((a + b) % p - c) % p = ((a - c) % p + (b * c) % p) % p
