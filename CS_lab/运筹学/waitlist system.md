- 条件 : 服务能力的限制

> input -> list -> system

### rule
FCFS(队列序) / LCFS(栈序) / PS(根堆序)

## 柏松过程
> 令N(t)表示[0,t]内到达系统的服务数,{N(t) , t>=0}是其随机过程
> 若该过程为柏松过程,则需满足如下三个条件:
> - 事件发生k次的概率只与区间长度有关
> - 不重叠的事件端相互独立
> - 充分小的区间内,超出的概率微乎其微

性质: 
$$P\{N(t) = n\} = (at)^n / n! * e^-at$$

- 
$$P_{n+1} = a_n / b_{n+1} * p_n$$
